Update guide:

1. Extract the downloaded file from codecanyon.

2. There you will get a zip folder named ‘updates.zip’. Upload that to the root directory on your server in which your previous version is running. Right Click on the zip file. Unzip that "updates.zip" file by selecting “Extract here”.

3. Now go to the home page of your website and click on ‘Update Now’.

4. Update will be completed!
